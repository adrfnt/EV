Para criar um doc HTML é necessário seguir os passos abaixo:

  1. Acessar o bloco.
  2. Indicar atravês da tag o tipo de documento criado.

Esses são os primeiro passos para se criar uma página web.

Agora, uma lista não ordenada:

  • Exemplo 1
  • Exemplo 2
  • Exemplo 3
  • Exemplo 4

Desta forma, para listas ordenadas utilize ol e para não ordenadas ul.

Exemplo de lista intercalada:

  1. Exemplo 1
    • Item 1
    • Item 2

  2. Exemplo 2
    • Item 1
